# Sales Performance Analysis (SQL)

## Objective
Analyze sales data to understand revenue trends, customer behavior, and profitability.

## Dataset
- Order-level sales data (CSV)
- Fields include order ID, amount, profit, quantity, category, customer, state, and date

## Key Analysis
- Total revenue and profit
- Orders and quantity trends
- Revenue by state and category
- Top customers by revenue
- Monthly revenue trends
- Profit vs loss classification

## Tools Used
- PostgreSQL
- SQL (aggregations, CASE, GROUP BY, date functions)

## Outcome
Generated insights that were later visualized in a Power BI dashboard.
