## Sales Performance Analysis Dashboard (SQL + Power BI)
## Project Overview

This project combines SQL querying and Power BI visualization to analyze sales performance across customers, categories, payment methods, and time. The dashboard helps identify revenue trends, customer value, and business performance indicators.

## Tools & Technologies

SQL (PostgreSQL / MySQL style queries)

Power BI

CSV / Excel

DAX Measures

## Key KPIs

Total Revenue: 6M

Total Profit: 2M

Total Orders: 547 (distinct count)

Total Quantity Ordered: 13K

Average Order Value (AOV): Auto-calculated

## Dashboard Insights

Revenue by payment method

Revenue by product category

Top customers by total revenue

Monthly revenue trends (MTD)

State-wise performance analysis

## SQL Logic Used

SUM() for revenue and quantity

COUNT(DISTINCT order_id) for total orders

GROUP BY and ORDER BY for aggregation

Date formatting for monthly trends

Writing analytical SQL queries

Data aggregation & filtering

Power BI KPI design

Time-series analysis

Business-focused insights
